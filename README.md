# Gerador e Receptor de Sinais FSK

Projeto da disciplina: Princípios de Simulação de Sistemas de Comunicação

## Objetivo

Simular a geração, transmissão e recepção de sinais FSK utilizando Python.

## Conceito

A modulação FSK (Frequency Shift Keying) representa bits digitais usando frequências diferentes:

* Bit 0 → frequência f0
* Bit 1 → frequência f1

## Etapas do código

1. Geração de bits aleatórios
2. Modulação FSK
3. Adição de ruído AWGN
4. Demodulação por correlação
5. Cálculo da taxa de erro (BER)

## Como executar

```bash
python fsk_simulacao.py
```

## Bibliotecas usadas

* numpy
* matplotlib
